# London Inn Menu

Estructura organizada para publicación web:

- `index.html`: Página principal
- `assets/css/`: Hojas de estilo
- `assets/js/`: Scripts JavaScript
- `assets/images/`: Imágenes del sitio
- `assets/icons/`: Íconos adicionales
- `assets/favicon/`: Favicons y manifest

## Instrucciones
1. Sube todo el contenido de la carpeta a tu hosting.
2. Asegúrate de que las rutas en `index.html` apunten a las carpetas `assets/`.
3. ¡Listo para publicar!
